![편집모드 활성화 버튼](/images/b2b-40.png) *편집모드 활성화 버튼*

편집모드 버튼은 [관리자](/docs/03%20부록/관리자%20계정)(`sales4`) 로그인 후 B2B 홈페이지 상단에서 켜고 끌 수 있습니다.

편집모드를 켜면, 위젯 설정 버튼 또는 게시글 별 설정을 바로 편집할 수 있는 버튼이 활성화되어 [관리자 설정](/docs/1%20B2B%20홈페이지%20운영/2.%20관리자%20설정)에 들어가지 않아도 각 기능 설정을 변경할 수 있습니다.

## 편집모드 활성화 후 메인화면

메인화면은 대문 위젯과 [관리자](/docs/03%20부록/관리자%20계정)로 로그인 시 나타나는 현황판으로 이루어져 있습니다.

대문 위젯의 텍스트 내용과 바로가기 버튼은 EB컨텐츠 마스터 설정 클릭 후 편집 가능합니다. 매뉴얼로 이동

## 편집모드 활성화 후 하단 바

하단 바는 [기업정보](/docs/1%20B2B%20홈페이지%20운영/2.%20관리자%20설정/9.%20테마설정관리#사업자-정보)와 [하단 링크], [2차 기업정보]로 이루어져 있습니다. (각각 클릭 시 해당 매뉴얼로 이동)

![편집모드 활성화 후 하단 바](/images/b2b-42.png) *편집모드 활성화 후 하단 바*

## 편집모드 활성화 후 게시판

게시판에서 편집모드를 활성화하면 게시판 레이아웃 상단에 **복제하기**, **기본설정**, **확장기능**, **확장필드** 설정 버튼이 나타납니다.

- **복제하기**: 현재 게시판을 복제하여 새로운 게시판을 만듭니다. (사용할 일 없음)
- **기본설정**: 현재 게시판의 기본설정 관리자 화면 창을 띄웁니다. [매뉴얼로 이동](/docs/1%20B2B%20홈페이지%20운영/2.%20관리자%20설정/3.%20게시판관리#기본-설정)
- **확장기능**: 해당 게시판의 확장기능 관리자 화면 창을 띄웁니다. (사용할 일 없음)
- **확장필드**: 해당 게시판의 확장필드 관리자 화면 창을 띄웁니다. [매뉴얼로 이동](/docs/1%20B2B%20홈페이지%20운영/2.%20관리자%20설정/3.%20게시판관리#확장필드)
