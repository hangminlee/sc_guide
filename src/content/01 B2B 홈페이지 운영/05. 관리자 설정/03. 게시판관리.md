![게시판관리 메뉴](/images/b2b-18.png) *게시판관리 메뉴*

게시판 관리 메뉴에서는 새로운 게시판을 추가하거나 게시판 이름을 수정할 수 있습니다.

## 게시판 추가

우측 상단의 **게시판 추가** 버튼을 누릅니다.

## 게시판 추가 / 설정 화면

게시판 추가 버튼을 누르거나 수정 버튼을 누르면 동일하지만 상황에 맞춰 약간 다르게 화면이 나타납니다.

### 기본 설정

![게시판 추가 / 설정 화면](/images/b2b-19.png) *게시판 추가 / 설정 화면*

- **TABLE**: 게시판의 고유코드 될 값입니다. 한글, 특수문자, 띄어쓰기 없이 입력합니다.
- **그룹**: 게시판 그룹입니다. 게시판이 해당 그룹에 속하게 됩니다.
- **게시판 제목**: 게시판의 제목입니다.
- **모바일 게시판 제목**: 모바일에서 접속할 시 표시될 게시판 제목입니다. 공란으로 두면 기본 게시판 제목이 출력됩니다.
- **게시판 스킨**: 기본 값인 **이윰빌더스킨**을 선택합니다.
- **접속기기**: PC와 모바일에서 모두 사용을 클릭합니다.
- **분류**: 카테고리입니다. 사용을 클릭하고 분류할 이름을 밑에 설명과 같이 `|` 문자(쉬프트 + 백스페이스 밑에 있는 `\`키)로 구분하여 입력합니다. 
- **글작성 후 이동 URL**: 비워둡니다.

### 권한 설정

![권한 설정 화면](/images/b2b-20.png) *권한 설정 화면*

게시판의 각종 기능에 대해 접근할 수 있는 권한을 설정합니다. 등급(`1`~`10`)을 설정하면 해당 등급 이상의 회원만 기능에 접근할 수 있습니다.

등급 숫자가 낮을수록 정보가 공개되며, 높을수록 해당 등급의 회원에게만 정보가 공개됩니다. 등급 `10`은 관리자 수준의 등급입니다.

- **게시판 관리자**: 게시판 관리자로 설정할 **아이디**를 입력합니다. `sang`은 대표님(`박상화`) 아이디이며, 게시판 관리자는 게시판에 등록된 게시글에 대해 메일로 알림을 받을 수 있으며, 비밀글이라도 조회가 가능합니다.

업체 전용 정보로 제한하려는 경우에는 일괄적으로 `3`으로 설정하면 됩니다.

### 기능 설정

세세한 기능 설정이 가능합니다만, 하나씩 설명하기에는 내용이 너무 방대합니다.

기능에 대한 설명을 확인하며 필요한 기능이 있는 경우 설정을 변경하여 직접 확인하면 될 것 같습니다.

### 디자인 / 양식

![디자인/양식](/images/b2b-21.png) *디자인 / 양식 설정 화면*

게시판 별 레이아웃 디자인(스킨)을 설정할 수 있습니다.

- **이윰 게시판 스킨**: 적용할 스킨을 선택합니다. 스킨은 서버에 업로드하면 사용할 수 있습니다.
- **상단 내용**: 여기에 내용을 입력하면 해당 게시판 상단에 내용이 표시됩니다.
- **하단 내용**: 여기에 내용을 입력하면 해당 게시판 하단에 내용이 표시됩니다.
- **이미지 폭 크기**: 게시글 내 삽입된 이미지가 설정된 수치로 화질이 변환되어 출력됩니다.

### 여분필드

![여분필드 설정 화면](/images/b2b-22.png) *여분필드 설정 화면*

그누보드는 기본 입력란 외에 각 게시글 데이터에 집어넣을 수 있는 추가 양식들을 설정할 수 있게 되어 있습니다. 이를 **여분필드**라고 하며 총 10개의 여분필드를 사용할 수 있게 되어 있습니다.

여분필드의 사용은 각 게시판 스킨 파일에서 `wr_1` ~ `wr_10`(필드값)까지의 값을 코드에 집어 넣는 것으로 사용할 수 있으며, 스킨 파일에서 각 필드값을 입력란에 할당해주지 않으면 사용할 수 없습니다.

- **여분필드 1** (wr_1): `예약업로드` - 게시판에 예약업로드 기능을 활성화한 경우 공개 시간이 여기 데이터에 기록됩니다.
- **여분필드 10** (wr_10): `끌어올리기 사용 여부` - 게시판 별 끌어올리기 기능을 사용하려면 여분필드 10 값에 `1`을 입력하고 저장합니다.

### 확장필드

![확장필드 설정 화면](/images/b2b-23.png) *확장필드 설정 화면*

확장필드는 그누보드의 기본 여분필드 이외에 입력란을 추가로 확장할 수 있는 기능입니다. 개수는 무제한이며, 확장필드 추가하기 기능을 통해 필드를 생성하고 각각 글 보기, 글 쓰기, 목록에서 확장필드를 사용할 수 있도록 예제 코드를 제공합니다.

#### 확장필드 추가 및 속성 편집

![확장필드 추가 화면](/images/b2b-24.png) *확장필드 추가 화면*

- **필드명칭**: 필드가 무엇을 위한 필드인지 설명하는 란입니다.
- **폼타입**: 입력란의 종류입니다. 원하는 입력란으로 선택하면 예제 코드가 그에 해당하는 예제 코드로 변경됩니다.
- **필수항목 여부**: 체크하면 예제 코드에 입력란을 강제하도록 코드가 추가 삽입됩니다.
- **필드종류**: 네가지 데이터 타입 중에서 선택할 수 있습니다. (`varchar`: 고정된 길이의 문자열 값 (지정 길이 이내의 문자열만 저장 가능), `char`: 고정된 길이의 문자열 값 (지정 길이의 문자열만 저장 가능), `int`: 고정된 길이의 2진수 정수값..., `text`: 길이 제한이 없는 문자열 값)
- **길이**: 길이를 설정하면 `text` 이외의 필드종류에서 길이를 제한합니다.
- **기본값**: 설정하면 필드가 비워져있을 때 해당 값이 기본값으로 들어가게 할 수 있습니다.

#### 예제코드 보기

추가된 확장필드의 각 줄에는 **코드보기** 버튼이 있습니다. 이를 클릭하면 예제코드를 볼 수 있습니다.

![코드보기 버튼](/images/b2b-25.png) *코드보기 버튼*

![코드보기 화면](/images/b2b-26.png) *코드보기 화면*

- **Write 코드**: 글쓰기 화면에 사용하려면 여기에 있는 코드를 삽입하면 됩니다. 단축코드를 사용하면 한줄로 코드를 삽입할 수 있으나, 세세한 설정을 원한다면 전체코드를 삽입하고 패턴을 파악해 원하는대로 수정하면 됩니다.
- **View 코드**: 글보기(본문) 화면에 사용하려면 여기에 있는 코드를 삽입하면 됩니다.
- **List 코드**: 글목록 화면에 사용하려면 여기에 있는 코드를 삽입하면 됩니다.
