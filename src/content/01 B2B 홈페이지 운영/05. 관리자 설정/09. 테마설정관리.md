기본 설정 이외의 사이트 설정, 주로 외관으로 보이는 문구 등을 설정할 수 있습니다.

기본 기능을 개조하여 영업팀 실정에 맞게 조정한 부분이 있기 때문에, 일부 항목은 여기에서 수정해도 적용되지 않는 경우가 있을 수 있습니다.


## 테마관리

별도로 건드릴 필요 없습니다.

## 홈페이지메뉴설정 (중요)

홈페이지 상단 메뉴를 설정하는 화면입니다.

![홈페이지 메뉴 설정 화면](/images/b2b-34.png) *홈페이지 메뉴 설정 화면*

### 현재 등록된 메뉴 패널

화면의 왼쪽에는 현재 등록된 메뉴가 표시되어 있습니다.[^눈-표시의-의미]

![현재 등록된 메뉴](/images/b2b-35.png) *현재 등록된 메뉴*

### 메뉴 수정 패널

#### 커뮤니티 메뉴를 클릭하거나 아무것도 클릭하지 않았을 때

**커뮤니티 메뉴**를 클릭하거나 아무것도 클릭하지 않으면 우측에 이와 같은 화면이 나타납니다.

- **대상 선택**: 메뉴링크를 직접 입력하고 싶으면 **직접입력**을 선택하고, 게시판 등 기존에 등록된 페이지를 연결하려면 연결하고자하는 대상을 선택합니다.
- **폰트어썸 아이콘**: 메뉴에 아이콘을 삽입하고 싶은 경우 [`Font Awesome`](/docs/03%20부록/Font%20Awesome)에서 제공하는 아이콘 값을 입력합니다. (버전 5)
- **메뉴명**: 화면에 노출될 메뉴명입니다.
- **메뉴링크**: 메뉴를 클릭하면 어디로 이동할지 입력합니다.
  - **현재창**: 메뉴를 클릭하면 새 탭이나 창을 열지 않고 현재 페이지에서 이동합니다.
  - **새창**: 메뉴를 클릭하면 새 탭이나 창을 열어 해당 링크를 표시합니다.
- **메뉴보이기 레벨 설정**: 해당 레벨 이상의 사용자에게만 해당 메뉴가 노출되도록 설정합니다.
- **메뉴 사용여부**: `사용안함`으로 설정하면 메뉴가 등록되지만 화면에 표시되지는 않습니다.
- **상단메뉴 사용여부**: `사용안함`으로 설정하면 해당 페이지 접속 시 상단 메뉴가 표시되지 않습니다.
- **사이드 레이아웃**: `사용안함`으로 설정하면 해당 페이지 접속 시 우측의 [사이드 레이아웃](/docs/03%20부록/사이드%20레이아웃)이 표시되지 않습니다.

![1차메뉴 생성 화면](/images/b2b-36.png) *1차메뉴 생성 화면*

### 메뉴수정 및 삭제

- **메뉴코드**: 고정값입니다.
- **출력순서**: 숫자가 작을수록 해당 메뉴가 먼저 화면에 보여집니다.
- **대상 선택**: 메뉴링크를 직접 입력하고 싶으면 **직접입력**을 선택하고, 게시판 등 기존에 등록된 페이지를 연결하려면 연결하고자하는 대상을 선택합니다.
- **폰트어썸 아이콘**: 메뉴에 아이콘을 삽입하고 싶은 경우 [`Font Awesome`](/docs/03%20부록/Font%20Awesome)에서 제공하는 아이콘 값을 입력합니다. (버전 5)
- **메뉴명**: 화면에 노출될 메뉴명입니다.
- **메뉴링크**: 메뉴를 클릭하면 어디로 이동할지 입력합니다.
  - **현재창**: 메뉴를 클릭하면 새 탭이나 창을 열지 않고 현재 페이지에서 이동합니다.
  - **새창**: 메뉴를 클릭하면 새 탭이나 창을 열어 해당 링크를 표시합니다.
- **메뉴보이기 레벨 설정**: 해당 레벨 이상의 사용자에게만 해당 메뉴가 노출되도록 설정합니다.
  - **하위메뉴에도 동일하게 적용하기**: 하위 메뉴가 있는 경우 하위 메뉴에 설정한 레벨과 무관하게 이 메뉴의 레벨 설정을 따라가게 합니다.
- **메뉴 사용여부**: `사용안함`으로 설정하면 메뉴가 등록되지만 화면에 표시되지는 않습니다.
- **상단메뉴 사용여부**: `사용안함`으로 설정하면 해당 페이지 접속 시 상단 메뉴가 표시되지 않습니다.
- **사이드 레이아웃**: `사용안함`으로 설정하면 해당 페이지 접속 시 우측의 [사이드 레이아웃](/docs/03%20부록/사이드%20레이아웃)이 표시되지 않습니다.
- **메뉴 삭제**: 선택한 메뉴를 삭제합니다.

![메뉴수정 및 삭제 화면](/images/b2b-37.png) *메뉴수정 및 삭제 화면*

### 하위메뉴 생성

선택한 메뉴의 하위 메뉴를 생성합니다.

현재 홈페이지는 최대 2단계의 하위메뉴를 표시할 수 있게 되어 있습니다. (기본 메뉴 > 하위 메뉴 > 하위 메뉴)

각 입력란의 설명은 [1차메뉴 생성 화면](#커뮤니티-메뉴를-클릭하거나-아무것도-클릭하지-않았을-때)과 동일하므로 해당 섹션을 참고해주세요.

![하위메뉴 생성 화면](/images/b2b-38.png) *하위메뉴 생성 화면*

## EB콘텐츠관리

B2B 홈페이지 곳곳에는 기본 레이아웃 이외에 별도로 배치해둔 위젯 같은 것들이 있습니다. 이것들이 대부분 **EB콘텐츠** 위젯입니다.

- **확인 방법**: 홈페이지 상단의 [편집모드](/docs/03%20부록/편집모드)를 활성화하면 각 위젯의 가장자리에 EB컨텐츠 편집 버튼이 나타납니다.

이미 추가된 각 위젯에 관해서는 [[관리자](/docs/03%20부록/관리자%20계정) 설정 (고급) 섹션]을 참고해 주시기 바랍니다.

![EB콘텐츠관리 화면](/images/b2b-39.png) *EB콘텐츠관리 화면*

### 좌측 패널

좌측 패널에는 현재 추가된 메뉴가 표시됩니다. 관리의 편의성으로 각 페이지 별로 EB콘텐츠 위젯을 추가해서 사용하라는 목적으로 존재하는 화면 같은데, 오히려 위젯이 저장된 위치를 찾기 힘들어질 가능성이 있어서 B2B 홈페이지에는 따로 메뉴를 지정해 위젯을 추가하지 않았습니다.

![EB콘텐츠관리 좌측 패널](/images/b2b-46.png) *EB콘텐츠관리 좌측 패널*

### 우측 패널

![EB콘텐츠관리 우측 패널](/images/b2b-47.png) *EB콘텐츠관리 우측 패널*

- [**EB콘텐츠마스터 추가**](#eb콘텐츠마스터-추가-관리): 새로운 위젯을 추가합니다.
- [**수정**](#eb콘텐츠마스터-추가-관리) (관리): 위젯의 속성 및 세부 설정을 수정합니다.
- [**아이템관리**](#-참고-eb콘텐츠-아이템관리-목록) (관리): 위젯에 구성요소를 추가하거나 할 수 있게 되어 있는 경우, 구성요소를 관리할 수 있습니다. (구성요소의 예: 메인화면의 바로가기 링크들)
- **콘텐츠마스터명**: 위젯의 이름으로 구분하기 위해 사용합니다.
- **치환코드**: 해당 위젯을 각 페이지 내에 삽입하기 위한 코드입니다. [소스코드](/docs/03%20부록/소스코드)에 직접 삽입하여 추가할 수 있습니다.
- **상태**: 위젯을 [소스코드](/docs/03%20부록/소스코드)에 추가했다고 하더라도 여기서 숨기기로 설정해 놓으면 해당 위젯이 표시되지 않습니다. 상태를 바로 변경하려면 상태를 설정하고 좌측 체크박스를 체크한 뒤에 `선택수정`을 눌러야 합니다.
- **선택수정** (목록 하단 좌측): 체크박스 선택된 항목의 수정 내용을 적용합니다.
- **선택삭제** (목록 하단 좌측): 체크박스 선택된 항목을 삭제합니다.

우측 패널에서는 위젯을 추가하거나 현재 추가된 위젯을 관리할 수 있습니다.

### EB콘텐츠마스터 추가 / 관리

EB콘텐츠 위젯을 새로 추가하거나 편집하는 화면입니다.

각 필드에 값을 입력하더라도 위젯에 따라 활용되는 값이 있을 수도 있고 없을 수도 있습니다. 커스텀 위젯을 만들어서 활용하려는 경우 내부 연결 값이 어떻게 되는지 출력하거나 디버깅하여 추가하여 활용할 수 있습니다.

- **코드**: 새로운 위젯을 추가하거나 기존 위젯을 편집할 때 표시되는 고유 번호입니다. 수정이 불가능합니다.
- **치환코드**: 치환코드복사를 누르거나 위 코드를 직접 드래그하여 복사한 뒤 원하는 코드 파일에 붙여넣으면 해당 위치에 위젯이 추가됩니다.
- **출력여부**: 치환코드를 코드 파일에 붙여넣어 배치했더라도 출력여부를 숨기기로 설정하면 해당 위젯은 표시되지 않습니다.
- **EB콘텐츠 마스터 명칭**: 해당 위젯을 설명하는 이름을 입력합니다.
- **EB콘텐츠 마스터 스킨**: 어떤 모양의 위젯을 배치할지 스킨을 선택하여 설정합니다. ([스킨 선택하기 화면](#-참고-스킨-선택하기-화면))
- **EB콘텐츠 타이틀 #1**: 해당 위젯에서 사용할 타이틀입니다.
- **EB콘텐츠 타이틀 #2**: 해당 위젯에서 사용할 보조 타이틀입니다.
- **EB콘텐츠 설명글 #1**: 해당 위젯에서 사용할 설명글입니다.
- **EB콘텐츠 설명글 #2**: 해당 위젯에서 사용할 보조 설명글입니다.
- **콘텐츠 아이템에서 사용할 링크수**: 해당 위젯에서 바로가기 링크를 사용하는 경우 링크 개수를 설정합니다. 추후 아이템 관리 화면에서 설정한 개수만큼 링크를 삽입할 수 있습니다.
- **콘텐츠 아이템에서 사용할 이미지수**: 해당 위젯에서 이미지를 사용하는 경우 이미지 개수를 설정합니다. 추후 아이템 관리 화면에서 설정한 개수만큼 이미지를 삽입할 수 있습니다.
- **콘텐츠 아이템에서 사용할 필드수**: 해당 위젯에서 입력 필드를 사용하는 경우 필드 개수를 설정합니다. 추후 아이템 관리 화면에서 설정한 개수만큼 입력 필드를 삽입할 수 있습니다.
- **대표 연결주소 [링크]**: 해당 위젯을 클릭하면 연결될 링크를 설정합니다.
- **EB콘텐츠 마스터 이미지**: 해당 위젯에서 사용할 마스터 이미지를 업로드합니다.
- **EB콘텐츠 마스터 첨부파일**: 해당 위젯에서 사용할 첨부파일을 업로드합니다.

![EB콘텐츠 추가 / 관리 화면](/images/b2b-48.png) *EB콘텐츠 추가 / 관리 화면*

#### (참고) EB콘텐츠 아이템관리 목록

EB콘텐츠 아이템관리 버튼을 클릭하면 추가된 위젯 내 항목들이 표시됩니다. (바로가기 링크 등)

각 항목 별 관리를 클릭하거나 EB콘텐츠 아이템 추가를 누르면 항목을 관리할 수 있습니다.

![EB콘텐츠 아이템 목록](/images/b2b-50.png) *EB콘텐츠 아이템 목록*

#### (참고) EB콘텐츠 아이템관리 추가 / 관리

각 아이템 별로 속성 값을 입력합니다. 위젯 자체와 마찬가지로 위젯에 따라 각 항목에 활용되는 값이 있을 수도 있고 없을 수도 있습니다.

- **게시여부**: 숨기기로 설정하면 추가된 항목이더라도 위젯에 표시되지 않습니다.
- **출력순서**: 항목의 출력 순서입니다.
- **보기권한 설정**: 권한 레벨을 설정하면 해당 레벨 이하 회원에게는 노출되지 않습니다.
- **노출 방식**: 기간제 방식으로 선택할 경우, 아래의 노출 기간에 해당하는 경우에만 해당 위젯이 노출됩니다.
- **노출 기간**: 기간제 방식으로 선택한 경우, 노출 기간을 설정합니다.
- **텍스트 필드 #1~#XX**: 위젯을 추가할 때 설정한 필드 개수만큼 텍스트 필드를 입력할 수 있습니다.
- **설명글 #1~#2**: 위젯 아이템 항목에서 사용할 설명글을 입력합니다.
- **내용**: 위젯 아이템 항목에 들어갈 내용을 입력합니다.
- **연결주소 [링크] #1~#XX**: 위젯 아이템 항목을 클릭할 때 연결될 바로가기 링크를 삽입합니다.
- **이미지 #1~#XX**: 위젯 아이템 항목에 표시할 이미지를 업로드합니다. 위젯을 추가할 때 설정한 이미지 개수만큼 설정할 수 있습니다.

## EB최신글관리

**EB최신글** 위젯은 게시판에 연동되는 위젯으로, 게시판에 최신글이 등록되면 해당 위젯에도 최신글로 표시됩니다.

- **확인 방법**: 홈페이지 상단의 [편집모드](/docs/03%20부록/편집모드)를 활성화하면 각 위젯의 가장자리에 EB최신글 편집 버튼이 나타납니다.

이미 추가된 각 위젯에 관해서는 [[관리자](/docs/03%20부록/관리자%20계정) 설정 (고급) 섹션]을 참고해 주시기 바랍니다.

- **[EB최신글 마스터추가](#eb최신글-마스터-추가-관리)**: 새로운 최신글 위젯을 추가합니다.
- **[수정](#eb최신글-마스터-추가-관리)** (관리): 추가된 최신글 위젯의 속성을 편집합니다.
- **치환코드**: 해당 위젯을 각 페이지 내에 삽입하기 위한 코드입니다. [소스코드](/docs/03%20부록/소스코드)에 직접 삽입하여 추가할 수 있습니다.
- **상태**: 위젯을 [소스코드](/docs/03%20부록/소스코드)에 추가했다고 하더라도 여기서 숨기기로 설정해 놓으면 해당 위젯이 표시되지 않습니다. 상태를 바로 변경하려면 상태를 설정하고 좌측 체크박스를 체크한 뒤에 `선택수정`을 눌러야 합니다.
- **선택수정** (목록 하단 좌측): 체크박스 선택된 항목의 수정 내용을 적용합니다.
- **선택삭제** (목록 하단 좌측): 체크박스 선택된 항목을 삭제합니다.

![EB최신글관리 화면](/images/b2b-52.png) *EB최신글관리 화면*

### EB최신글 마스터 추가 / 관리

EB최신글 위젯을 새로 추가하거나 편집하는 화면입니다.

- **코드**: 최신글 위젯이 추가될 때 자동으로 생성되는 고유 번호이며 수정이 불가능합니다.
- **치환코드**: `치환코드복사` 버튼을 누르거나 위 코드를 복사하여 원하는 페이지의 [소스코드](/docs/03%20부록/소스코드)에 붙여넣으면 해당 위치에 위젯이 삽입됩니다.
- **출력여부**: 위젯을 [소스코드](/docs/03%20부록/소스코드)에 추가했다고 하더라도 여기서 숨기기로 설정해 놓으면 해당 위젯이 표시되지 않습니다.
- **최신글마스터 제목**: 해당 위젯의 제목을 입력합니다.
- **최신글마스터 스킨**: 어떤 모습으로 최신글 위젯을 표시할 지 스킨을 선택합니다.
- **캐시 갱신 시간**: 설정한 주기에 맞추어 최신글 목록이 갱신됩니다.
- **새글 표시 시간**: 설정한 시간 이내에 작성된 글에 최신글 표시를 추가로 붙여줍니다.
- **대표 연결주소 [링크]**: 최신글 위젯의 제목을 클릭했을 때 연결될 페이지의 주소를 입력합니다.

![EB최신글 마스터 추가 / 관리 화면](/images/b2b-53.png) *EB최신글 마스터 추가 / 관리 화면*
